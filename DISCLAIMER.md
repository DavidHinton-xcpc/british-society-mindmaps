## English

This repository contains **interactive mind maps** created for the purpose of learning and reviewing British Society & Culture course material.

### Purpose

The mind maps in this repository are **study aids** — they are not intended to replace original textbooks or course materials. They were created as an independent, transformative educational resource to help students organize and review knowledge.

### Content Attribution

The knowledge framework is based on a British Society & Culture university course syllabus, covering the following chapters:

| Chapter | Topic |
|---------|-------|
| Ch1 | General Introduction |
| Ch2 | A Brief History (I) |
| Ch3 | A Brief History (II) |
| Ch4 | Political System |
| Ch7 | Education System |
| Ch10 | News Media |

The original source Markdown files are included in the `src/` directory for reference.
